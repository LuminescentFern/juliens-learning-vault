# Conducting a population census

This lecture explains methods ecologists use to estimate the population size of animals in a given area.

Population Counting Techniques

- Counting alligators can be done by shining a flashlight at night to spot their glowing eyes or by counting nests to estimate the number of females.
- These methods work well for large, visible animals like alligators but are less effective for elusive species.

Mark-Recapture Method

- For reclusive animals like venomous snakes, ecologists use the mark-recapture technique: capture individuals, mark them safely, release them, then recapture to see how many marked individuals are found.
- The population size is estimated using a formula based on the number of animals caught in the first and second captures and how many were recaptured.

Using Unique Markings and Camera Traps

- Some animals, like bobcats, have unique natural markings that allow identification without physical marking.
- Camera traps with motion detectors capture photos, and comparing spot patterns helps estimate population size by identifying individual animals.
